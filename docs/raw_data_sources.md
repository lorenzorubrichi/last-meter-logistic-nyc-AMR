# Raw Data Sources

The repository includes small schema-compatible sample extracts in `data/raw/`.
For full reproduction, replace them with complete datasets from the original
sources.

## Required Inputs

| Local filename | Description | Source link |
| --- | --- | --- |
| `BUILDING.csv` | NYC building footprint / building attributes | https://opendata.cityofnewyork.us/ |
| `Centerline.csv` | NYC street centerline geometries | https://opendata.cityofnewyork.us/ |
| `nyc_neighborhoods.geojson` | Neighborhood / NTA boundary layer | https://opendata.cityofnewyork.us/ |
| `pluto_full.csv` | PLUTO land-use and building attributes | https://opendata.cityofnewyork.us/ |
| `AddressPoint_full.csv` | Address point coordinates | https://opendata.cityofnewyork.us/ |
| `population.csv` | Population by neighborhood/NTA | https://opendata.cityofnewyork.us/ |
| `traffic.csv` | Traffic context data | https://opendata.cityofnewyork.us/|
| `pedestrian_mobility.csv` | Pedestrian mobility / corridor category | https://opendata.cityofnewyork.us/ |
| `NYC_Sidewalk.csv` | Sidewalk geometries | https://opendata.cityofnewyork.us/ |
| `Parking_Regulation.csv` | Parking regulation signs | https://opendata.cityofnewyork.us/ |
| `Parking_Meters.csv` | Parking meter locations/status | https://opendata.cityofnewyork.us/ |
| `Parking_Meters_-_ParkNYC_Block_Faces.csv` | ParkNYC blockface context | https://opendata.cityofnewyork.us/ |
| `Parking_Meters_-_Citywide_Rate_Zones.csv` | ParkNYC rate zones | https://opendata.cityofnewyork.us/ |
